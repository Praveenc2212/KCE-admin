# Project Overview

This is the Admin Project for the College Leave Management System.
It shares a MongoDB database with the main Leave Portal project but focuses strictly on **master data management**.

## Responsibilities
- Manage Student
- Manage Staff
- Department Management
- Security Management
- Tutor Mapping
- HOD Mapping

## Out of Scope
The Admin Panel **DOES NOT** manage:
- Leave records
- Outpass records
- Approvals
- Tracking dashboards
